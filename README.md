# homebrew-tools

This repository contains various tools for Homebrew.

## Usage

Tap the repository: `brew tap eendroroy/tools`
Install formula/cask `brew install [--cask] eendroroy/tools/<formula>`

## Available Formulas

*   **[gitter-rs](https://github.com/eendroroy/gitter-rs)**: `brew install eendroroy/tools/gitter-rs`
